# 动态规划（分阶段求解问题，父问题的结果由求解子问题得到）

问题建模：最优子结构、边界、状态转移公式  
求解问题：递归求解、备忘录算法（记录下每次算的结果，避免重复运算）、动态规划（自底向上，先求解子问题）

以**斐波那契数列**为例:
```
F(0) = 0
F(1) = 1
F(n) = F(n-1) + F(n-2) n>=2
```

Q: 求解 F(10)  
A:   
最优子结构：F(2) = F(1) + F(0)  
边界：F(1) 和 F(0)  
状态转移公式：F(n) = F(n-1) + F(n-2)
